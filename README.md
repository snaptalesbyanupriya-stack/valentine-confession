## Setup Instructions

### 1. Update Password
Open `index.html` and change the password (currently set to `0506`):

```javascript
const PASSWORD = "0506"; // Change this to your special date!
```

### 2. Add Your Photos
Replace these image filenames in the `photos` array with your own:

```javascript
const photos=[
  "[IMG_0030](https://drive.google.com/drive/u/0/folders/13voRyw6NYu3-kSrwEskewgY9Lz4HJ5Cq).jpeg",
  "[IMG_7246.jpeg](https://drive.google.com/drive/u/0/folders/15Uyb9vrAiB6qurIYvH1Art6ILcVx4cLF)",
  "[IMG_8016.jpeg](https://drive.google.com/drive/u/0/folders/15Uyb9vrAiB6qurIYvH1Art6ILcVx4cLF)",
  "[IMG_9930.jpeg](https://drive.google.com/drive/u/0/folders/1usBm5QVNSxIK0hSfw4ZQ68RNZrGlxad_)"
];
```

Place your image files in the same directory as `index.html`.

### 3. Deploy with GitHub Pages

1. Go to your repository **Settings**
2. Scroll to **GitHub Pages** section
3. Select **main** branch as the source
4. Your site will be live at: `https://snaptalesbyanupriya-stack.github.io/valentine-confession/`
